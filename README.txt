WooCommerce Plugin to generate a Landing Page foreach Coupon

== Description ==

This plugin will help you to generate a landing page for each Coupon you've created in WooCommerce.
This is really useful to share a coupon on Social Networks.

== Installation ==

1. Install `WooCommerce`
2. Upload `coupon-landing-page.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
